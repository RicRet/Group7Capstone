# This is Eagle Guide
## For our dev:

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
      npx expo start --tunnel
   ```
Must run with --tunnel in order to be able to load application in other device

3. Open in web
   once project is loaded please load once in web, before attempting to use QR code
